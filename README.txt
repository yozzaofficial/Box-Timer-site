# Boxe Timer

**Boxe Timer** is a simple web app designed for anyone who wants to practice boxing with timed rounds and rest intervals. It provides a minimal and responsive interface, suitable for mobile and desktop use.

## About

This project was created both as a personal coding exercise and a practical tool for boxing workouts. The goal was to build a lightweight timer app with a clean UI and a responsive design.

## Features

- Mobile-first responsive layout
- Burger menu that opens when scrolling to the top
- Smooth scrolling animations
- Basic device adaptability via media queries

## Technical Approach

- Built with HTML, CSS, and TypeScript (vanilla)
- Mobile-first design strategy
- Custom TypeScript for menu behavior
- Form with validation 
- Planned improvements: script optimization, better code organization, inline comments
